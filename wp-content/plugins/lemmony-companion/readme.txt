== Description ==

The Lemmony Companion is a companion plugin for the Lemmony theme. Adds additional block editor blocks needed for the best theme experience.

== Installation ==

1. Go to Plugins > Add New, search for "Lemmony Companion" and then click "Install Now".
2. After installaion is done click on "Activate".
